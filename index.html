Welcome to my Console App! Learn about Visual Studios!

To get Visual Studios, go to https://code.visualstudio.com/Download and download on mac or windows. Next open up Visual Studios and go open a folder. Save the folder as any name you want and open it in Visual Studios. Then, click and press new file. MAke sure this file is inside the folder, and then change it to index.html. Next, start coding!



The console screen is inside developer tools and you can veiw your uotput of code there. The console screen is very helpful and fun to use with coding. To go to it, right click your screen after you run it on chrome, and then hit Inspect. There, go to the top and next to elements, click Console. To print things on console, the code is: console.log();
